# 斗地主助手

一个基于iOS巨魔工具的斗地主AI助手，用于单机斗地主游戏。

## 功能特性

- 🎮 简洁的几何风格UI，一键启动
- 🪟 全局悬浮窗显示AI建议
- 🤖 AI智能分析三个阶段：叫地主、加倍、出牌
- 📸 自动OCR识别游戏画面
- 🔄 后台运行不被杀死

## 构建说明

本项目使用GitHub Actions自动构建，无需Mac和Xcode。

1. Fork本仓库
2. 推送代码到main分支
3. GitHub Actions自动构建
4. 下载生成的deb文件

## 安装

1. 下载构建好的deb文件
2. 使用巨魔助手安装
3. 打开应用，点击启动按钮

## 技术栈

- Objective-C
- Theos
- 百度OCR API
- 斗地主AI API
